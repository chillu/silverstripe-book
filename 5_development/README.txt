# Installation

- Set this folder as the webroot in your webserver, it will usually be accessible as http://localhost/
- Please import the provided database.sql *after* running the web-based installer
- Copy relevant configuration from mysite/_config_example.php to mysite/_config.php
- Ensure that your theme is set to "mysite" in mysite/_config.php
- The CMS interface is available at http://localhost/ (User: admin, Password: password)
